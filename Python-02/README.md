# Python 02 — Exercises

Collection of Python exercises and small programs grouped by topic.

How to run
- Use Python 3: `python script.py`.

Subfolders
- `Dictionary` — examples and exercises using Python dictionaries
- `Functions` — function-focused exercises
- `not completed` — work-in-progress scripts
