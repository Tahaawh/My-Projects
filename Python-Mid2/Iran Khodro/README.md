# سیستم مدیریت ایران خودرو

برنامه‌ای برای مدیریت فروش خودروهای ایران خودرو با رابط کاربری تحت پنجره (GUI) ساخته شده با `CustomTkinter`.

## 📋 ساختار پروژه

```
Iran Khodro/
├── main.py              # برنامه اصلی
├── backend.py           # بانک داده و منطق
├── auth_page.py         # صفحات ورود و ثبت‌نام
├── admin_panel.py       # پنل مدیریت ادمین
├── user_panel.py        # پنل کاربر
├── users.json           # فایل کاربران
├── cars.json            # فایل خودروها
├── comments.json        # فایل کامنت‌ها
└── README.md           # این فایل
```

## 🎯 ویژگی‌های برنامه

### 🔐 احراز هویت
- **ورود**: ورود با نام کاربری یا کد ملی
- **ثبت‌نام**: ثبت نام کاربر جدید با اعتبارسنجی رمز عبور

### 👨‍💼 پنل ادمین
- **مدیریت خودروها**: افزودن، حذف خودروها
- **مدیریت کاربران**: حذف کاربران، ارتقا به VIP

### 👤 پنل کاربر
- **مشاهده خودروها**: فهرستی از تمام خودروها
- **سبد خرید**: افزودن/حذف خودروها
- **خرید**: ثبت خرید و ذخیره در تاریخچه
- **پروفایل**: ویرایش اطلاعات و تاریخچه خرید
- **کامنت**: کامنت‌گذاری روی خودروها

## ⚙️ نیازمندی‌ها

```bash
pip install customtkinter
```

## 🚀 راه‌اندازی برنامه

```bash
python main.py
```

## 👤 حساب‌های نمونه

### حساب ادمین
- **نام کاربری**: `admin`
- **رمز عبور**: `Admin@123`

## 📝 اعتبارسنجی رمز عبور

رمز عبور باید شامل:
- ✓ حداقل 8 کاراکتر
- ✓ حداقل یک عدد
- ✓ حداقل یک حرف انگلیسی
- ✓ حداقل یک کاراکتر خاص (`!@#$%^&*()...`)

## 🏗️ معماری کد

### `main.py`
برنامه اصلی و مدیریت پنجره‌های مختلف

### `backend.py`
- `DataStore`: کلاسی برای مدیریت بانک داده
- توابع کمکی برای تاریخ و ساعت

### `auth_page.py`
- `AuthPage`: کلاسی برای مدیریت صفحات ورود و ثبت‌نام

### `admin_panel.py`
- `AdminPanel`: کلاسی برای مدیریت پنل ادمین

### `user_panel.py`
- `UserPanel`: کلاسی برای مدیریت پنل کاربر
